# 🎮 Matrix Flash Shooter

> A mind-bending, retro-futuristic shooter game where you blast through cascading matrices in a neon-drenched digital universe

## 🌟 Overview

**Matrix Flash Shooter** is a cutting-edge implementation of a fast-paced arcade shooter that combines the iconic visual aesthetic of the Matrix with modern web technologies. Navigate through streams of falling code, identify threats in real-time, and eliminate targets before they breach your defenses.

## ✨ Features

- **🔢 Dynamic Matrix Rain**: Experience authentic cascading code effects with customizable characters and colors
- **⚡ Lightning-Fast Gameplay**: React-based rendering for 60+ FPS smooth performance
- **🎯 Precision Targeting**: Advanced hit detection using modern collision algorithms
- **🌈 Visual Effects**: GPU-accelerated particle systems and screen-space effects
- **🎵 Immersive Audio**: Spatial audio with Web Audio API for an immersive experience
- **📊 Score System**: Real-time leaderboards with persistent data storage
- **🎨 Customizable Themes**: Multiple color schemes and visual presets
- **📱 Responsive Design**: Play seamlessly across desktop, tablet, and mobile devices

## 🚀 Modern Tech Stack

This project will leverage the latest and greatest in web development:

### Core Technologies
- **Frontend Framework**: React 18+ with Hooks and Concurrent Features
- **Build Tool**: Vite for lightning-fast builds and hot module replacement
- **Language**: TypeScript for type-safe development
- **Canvas Rendering**: HTML5 Canvas API with WebGL for 3D effects

### State & Logic
- **State Management**: Zustand for lightweight, scalable state management
- **Animation**: Framer Motion for smooth transitions and effects
- **Physics**: Custom physics engine with delta-time calculations

### Styling & UI
- **CSS Framework**: Tailwind CSS for rapid UI development
- **Component Library**: Radix UI for accessible, unstyled components
- **Icons**: Lucide Icons for crisp, customizable icons

### Development Tools
- **Code Quality**: ESLint + Prettier for consistent code style
- **Testing**: Vitest for unit tests, Playwright for E2E testing
- **CI/CD**: GitHub Actions for automated testing and deployment
- **Package Manager**: pnpm for efficient dependency management

## 🎯 Game Concept

In **Matrix Flash Shooter**, you're a digital guardian tasked with protecting the mainframe from malicious code injections. As streams of green characters cascade down your screen, you must:

1. **Identify** hostile code patterns among benign matrix rain
2. **Target** threats with precision before they reach critical system levels
3. **Eliminate** malware with powerful digital weapons
4. **Survive** increasingly complex attack patterns across multiple levels

### Game Modes
- **Campaign**: Progress through story-driven missions
- **Survival**: Endless waves of increasing difficulty
- **Time Attack**: Maximum score within time limits
- **Zen Mode**: Relaxed gameplay for casual players

## 🎨 Visual Style

Inspired by the iconic Matrix aesthetic, the game features:
- Authentic green-on-black terminal styling
- Glowing neon accents and particle effects
- CRT screen simulation with scanlines and phosphor glow
- Japanese katakana and custom glyphs
- Depth-of-field effects for enhanced atmosphere
