# 🎮 Matrix Flash Shooter

> A retro-futuristic shooter game where you blast through cascading matrices in a neon-drenched digital universe

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Status](https://img.shields.io/badge/status-in%20development-orange.svg)

## Overview

A fast-paced arcade shooter combining Matrix-style visual aesthetics with modern web technologies. Navigate through streams of falling code, identify threats in real-time, and eliminate targets before they breach your defenses.

## Key Features

- Dynamic Matrix rain effects with authentic cascading code
- Lightning-fast gameplay with React rendering (60+ FPS)
- Precision targeting and advanced collision detection
- GPU-accelerated particle effects and visual polish
- Multiple game modes: Campaign, Survival, Time Attack, Zen
- Responsive design for desktop, tablet, and mobile

## Tech Stack

- **Frontend**: React 18+ with TypeScript
- **Build**: Vite with hot module replacement
- **Styling**: Tailwind CSS
- **State**: Zustand
- **Testing**: Vitest & Playwright
- **Canvas**: HTML5 Canvas API with WebGL
